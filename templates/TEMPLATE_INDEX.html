PPT AGENT / TEMPLATE INDEX

模板预览索引页

这不是一组一次性的成品页，而是一组可复用的 SVG 母版。后续做 PPT 时，优先先从这里筛 3 个方向，再生成候选页让用户选。

使用顺序

1. 先确认主题与风格草案。

2. 再从模板库中筛 3-6 个最匹配的母版。

3. 基于其中 3 个做候选页，不从空白画布硬起稿。

4. 用户选定后，再按模板骨架扩成整套，并补结构图、渐变和质感层。

筛选建议

  • 开场/章节：优先选 `Dark Signal Opening` 或 `Story Chapter Break`。
  • 核心判断：优先选 `Core Conclusion Banner`。
  • 人群/场景：优先选 `Users Scenarios Split`。
  • 增长/闭环：优先选 `Growth Flywheel Signal`。
  • 风险/限制：优先选 `Risks Strip Grid`。
  • 方法/流程：优先选 `Process Compiler Pipeline`。
  • 论证/汇报：优先选 `Consulting Split Argument`。
  • 指标/数据：优先选 `Bento Metric Dashboard`。
  • 迁移/分类：优先选 `Quadrant Pattern Map`。
  • 结尾/原则：优先选 `Summary Manifesto`。

Bento Metric Dashboard
